trx is a simple desktop tool for turning audio and video into text. It uses Whisper for speech to text, which helps you make captions, subtitles, and transcripts from common media files.

This guide shows you how to get trx on Windows and start using it with no coding needed.

trx helps you:

## 🧭 Tips for Better Results

For cleaner transcripts:

- use clear audio
- keep one person speaking at a time when possible
- reduce background noise
- trim dead air before long recordings
- use the right language setting if the app asks for it

If a file has music under speech, the transcript may need a quick edit after the run.

## 🔧 Common Problems

### App will not open

- make sure you downloaded the Windows build
- check that the file is fully extracted
- try a fresh download from the Releases page

### File does not load

- check that the file type is supported
- move the file to a simple folder path like `C:\trx\media`
- avoid long folder names with special characters

### Transcript looks wrong

- check the audio quality
- use a cleaner recording
- try a file with less background noise
- split long recordings into smaller parts

### Output file is missing

- look in the same folder as your source file
- check the output path in the app
- make sure you have write access to that folder
